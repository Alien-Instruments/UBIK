Welcome to Alien Instruments UBIK, the talking MIDI mapping solution. Create groups for your controls and start adding sliders and buttons to send MIDI CC’s. Completely accessible for screen reading and keyboard navigation. Labelling groups and components also labels for screen reading. Design your own interface and map it to the device or software you wish to control. Each slider has a MIDI learn button, click this and follow the voice prompts to map your controller. When adding a slider, you can map the output range displayed on the screen and spoken aloud to an integer or float to match the parameter you are controlling. Alternatively, you can display or hear the current MIDI CC value. For buttons you can set the on value to any cc value, when mapped this will speak as “parameter name” on and off and display the same in the button. To get started click the add group button and follow the prompts. Or take a guided tour of the interface and settings by clicking the "Start Tour" button.

If you are using a screen reader to take the tour after clicking start tour focus will go to a heading level 2, this is the parameter name, and that component shows focus in the UI. Use the tab key and next will be a heading level 3 with a description and context of the area or item. Press tab again and you will focus on the next button, followed by previous and then close buttons. When the tour ends there will be a finish button.

Have fun and enjoy!
